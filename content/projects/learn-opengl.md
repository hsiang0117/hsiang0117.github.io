## LearnOpenGL Notes

[GitHub](https://github.com/hsiang0117/LearnOpengl)

学习 [LearnOpenGL.com](https://learnopengl.com/) 的代码仓库，包含完整的示例代码和个人注释。

### 涵盖的主题

| 章节 | 内容 |
|------|------|
| Getting Started | 窗口创建、三角形、着色器、纹理、变换、坐标系统、摄像机 |
| Lighting | 颜色、基础光照、材质、光照贴图、光源类型、多光源 |
| Model Loading | Assimp 模型加载、网格、模型类 |
| Advanced OpenGL | 深度测试、模板测试、混合、面剔除、帧缓冲、立方体贴图、高级数据、高级 GLSL |
| Advanced Lighting | 阴影映射、点光源阴影、法线贴图、视差贴图、HDR、Bloom、延迟着色、SSAO |
| PBR | 理论、IBL 漫反射、IBL 镜面反射 |

### 环境

- OpenGL 4.6 + GLSL 4.60
- GLFW3 + GLAD
- C++ 17
