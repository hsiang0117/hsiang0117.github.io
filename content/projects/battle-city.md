## BattleCity

[GitHub](https://github.com/hsiang0117/BattleCity) ⭐ 1

经典 FC 红白机游戏《坦克大战》(Battle City) 的 Java 复刻。

### 功能

- 🎮 **单人 & 双人模式** — 经典双人对战
- 🗺️ **自定义地图** — 支持读取地图文件
- 🔊 **音效系统** — 射击、爆炸、移动音效
- 🎨 **像素级渲染** — 还原经典视觉风格

### 技术细节

- **语言**: Java
- **GUI**: Java Swing
- **架构**: MVC 模式，游戏循环 60 FPS
